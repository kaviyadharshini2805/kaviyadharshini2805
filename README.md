# Featured Projects



## Candidate Intelligence Platform



A production-grade candidate data ingestion, normalization, and merging platform built as part of the Eightfold Engineering Internship assignment.



### Key Features



- Multi-source candidate data ingestion

- Smart duplicate detection

- Conflict resolution using configurable priority rules

- Data provenance tracking

- Runtime configurable output generation



**Tech Stack:** Python, Streamlit, Pydantic, Pandas



---



## Personal AI Chatbot



An AI-powered chatbot that answers questions about my skills, projects, education, and experience using Retrieval-Augmented Generation (RAG). The chatbot performs semantic search over my portfolio using ChromaDB and generates context-aware responses using Google's Gemini model.



**Tech Stack:** Python, Streamlit, LangChain, ChromaDB, Hugging Face Embeddings, Google Gemini

## File Organizer with MCP



An intelligent file management application that organizes files automatically using the Model Context Protocol (MCP). Users can organize files through natural language instructions and configurable automation rules.



**Tech Stack:** Python, MCP
